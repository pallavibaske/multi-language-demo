# 🌍 Multi-Language Demo

## 🎯 Project Overview

This project is a simple Python application that converts numbers into different languages. It demonstrates basic Python programming concepts, including functions, dictionaries, and conditional statements.

---

## ✨ Features

* Convert numbers into multiple languages
* Simple Python implementation
* Easy to understand and modify
* Beginner-friendly project

---

## 🛠️ Technologies Used

* Git
* GitHub

* AWS CodeBuild (buildspec.yml)
